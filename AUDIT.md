## 🔴 КРИТИЧНО — блокируют продакшн

- [ ] **HTTPS/TLS** — весь трафик HTTP, WebRTC требует HTTPS
- [ ] **ws:// → wss://** — LiveKit должен работать по wss://
- [ ] **CORS = \*** — разрешает запросы с любого сайта
- [ ] **Unbounded thread pool** — thread::spawn на каждый запрос → OOM при нагрузке
- [ ] **Данные только в памяти** — комнаты теряются при перезапуске

## 🟠 ВЫСОКИЙ

- [ ] Rate limiting на /api/token и /api/rooms
- [ ] HTTP Security headers (CSP, X-Frame-Options, X-Content-Type-Options)
- [ ] API key виден в stdout при старте
- [ ] WS URL захардкожен — добавить /api/config
- [ ] Нет graceful shutdown
- [ ] Нет мониторинга и алертов
- [ ] Нет ротации логов
- [ ] unwrap() на системное время и Mutex
